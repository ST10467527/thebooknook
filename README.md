1. Project Overview

The Book Nook website was developed to provide a functional, accessible, and visually appealing online platform for a real-world independent bookstore based in Johannesburg, South Africa.

The website aims to strengthen the business’s digital presence, improve communication with customers, and increase engagement through online interaction, event promotion, and product showcasing.

The site is structured using HTML5 and CSS3 with a focus on usability, mobile responsiveness, and simplicity. It reflects professional website development practices suitable for small businesses entering the digital market.

2. Project Objectives

The main objectives of this project are:

To design a user-friendly, responsive website that functions across all devices.

To create a professional online presence that communicates The Book Nook’s brand identity.

To provide customers with clear navigation to browse books, view events, and contact the store.

To use accessible web design principles to ensure inclusivity.

3. Key Performance Indicators (KPIs)

The success of this website will be measured by:

KPI	Measurement Method
User Engagement	Website traffic analytics (visits, time spent per page)
Customer Interaction	Number of enquiries received through the Contact Form
Sales Conversions	Number of purchases or store visits following website referrals
Event Participation	Sign-ups for workshops, launches, and reading events
Mobile Usability	Percentage of visitors using mobile devices successfully accessing content

5. Design Features & User Experience
Feature	Description
Responsive Navigation Bar	Adjusts between inline (desktop) and stacked (mobile). Includes dropdown menu for subpages.
CSS Pseudo-Classes	:link, :visited, :hover, :active, and :focus used for interactive behaviour and accessibility.
Hero Banner	Engaging homepage introduction with call-to-action button.
Semantic HTML5 Tags	header, main, section, footer, and nav replace unnecessary <div> tags.
Accessibility Features	ARIA roles and proper image alt-texts for screen readers.
Footer Contact Section	Quick contact details and store information across all pages.
